# Tic Tac Toe
The well-known Tic Tac Toe game built using OOP. It is built as a player vs player console application.  

## How to Run

1. Clone this repository:
```
git clone <repo-link>
```
2. Run the cpp files:
```
g++ *.cpp && .\a.exe
```
